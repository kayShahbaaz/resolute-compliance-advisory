## Engagement Overview

This engagement is a delta assessment of the Al Ameen insurance organisation's progress against the updated Saudi National Cybersecurity Authority Essential Cybersecurity Controls framework (NCA ECC-2:2024). This assessment builds upon the original NCA ECC-1:2018 audit (RCA-P3) and evaluates conformance against the newly released ECC-2:2024 framework, which introduces enhanced controls and updated requirements.

### Assessment Scope

- **Delta Assessment** – Comparative evaluation of ECC-1:2018 vs. ECC-2:2024
- **Control Updates** – Analysis of new, modified, and deprecated controls
- **Remediation Progress** – Review of corrective actions from original audit
- **SAMA CSF Maturity** – Updated maturity assessment

## Framework Evolution: ECC-1:2018 → ECC-2:2024

### What Changed in ECC-2:2024

The updated NCA ECC-2:2024 framework introduces:

- **Enhanced Governance Requirements** – Formal cybersecurity strategy and board oversight
- **Zero Trust Architecture** – Updated access control principles
- **Supply Chain Security** – Extended third-party and vendor security requirements
- **Cloud Security** – New controls for cloud infrastructure security
- **API Security** – Controls for application programming interface protection
- **Artificial Intelligence & Machine Learning** – Security considerations for AI/ML systems
- **Data Residency** – Enhanced data localization requirements
- **Incident Response Maturity** – Elevated incident detection and response capabilities
